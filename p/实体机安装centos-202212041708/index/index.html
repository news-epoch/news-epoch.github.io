2022-12-12
阅读时长: 3 分钟

准备工作

1 下载不同版本的centos

  1. 开启魔法上网
  2. 访问7.2.1511_isos_x86_64 (centos)
  3. 下载

2 windows安装UltraISO

  1. 下载：UltraISO - ISO CD/DV download page
  2. 安装软件

2.1 刻录U盘为启动盘

  1. 打开准备好的ISO镜像
  2. 然后再次回到软碟通（UltraISO）的窗口，点击菜单栏中的“启动”选“写入硬盘镜像”
  3. 接下来在弹出的窗口直接点击“写入”按钮（注：将写入方式选择为“USB-HDD+”，如果不是这个模式，可能导致有些电脑无法通过U盘正常启动。）

2.2 刻录CD/DVD为启动盘

  1. 选择刻录光盘映像
  2. 默认选项，开始刻录

开始安装

1 重启服务器到bios，并且设置为U盘启动或者CD/DVD启动

2 安装系统

  1. 进入这个界面，键盘输入 E 或者是tab
  2. 将CentOS\x207\x20x86_64 修改为**/dev/挂载的U盘名字** 或者**/dev/cdrom**这个是挂载的CD/DVD
  3. 正常接下来就是按序操作

title: 注意如果安装server with GUI模块时，一直卡顿在安装软件包的界面上
1. **可以选择最小安装，然后在进入到系统里面后，挂载yum本地源，输入`yum groupinstall "SERVER WITH GUI"`**
```yum.repo
[Local]
name=CentOS-$releasever - Local
baseurl=file:///media/cdrom
enabled=1
gpgcheck=0
```
2. `systemctl set-default graphical.target` (现在默认启动图形界面设置好了)
3. `systemctl set-default multi-user.target` (默认启动字符界面设置好了)
4. `shutdown -r now`

title: 当电脑存在第二块硬盘并且不想格式化
1. 安装软件包  `wget http://tuxera.com/opensource/ntfs-3g_ntfsprogs-2011.4.12.tgz`
2. 解压压缩包 `tar -zxvf ntfs-3g_ntfsprogs-2011.4.12.tgz`
3. 进入文件夹`cd ntfs-3g_ntfsprogs-2011.4.12`
4. 运行`yum -y install gcc`
5. 运行`./configure`
6. 运行`make`
7. 运行`make install`
8. 运行`mount -t ntfs-3g 挂载磁盘   目录`

title: 释放home空间，扩展到根目录上
```Linux
1. 备份home分区文件
2. umount /home
3. lvremove /dev/mapper/centos-home
4. lvextend -L +40G /dev/mapper/centos-root
5. xfs_growfs /dev/mapper/centos-root
6. lvcreate -L 1G -n /dev/mapper/centos-home
7. lvcreate -L 1G -n /dev/mapper/centos-home
8. mount -a
```
