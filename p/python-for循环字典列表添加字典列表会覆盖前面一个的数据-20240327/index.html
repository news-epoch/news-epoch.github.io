2 正确代码

如果你在使用循环遍历字典列表时，每次都添加一个新的字典列表，并且发现新添加的字典列表会覆盖前面一个的数据，那么可能是因为你在循环内部没有创建新的字典。 在循环内部，如果你使用相同的字典进行添加操作，那么每次添加的实际上是同一个字典对象的引用，导致后续的修改会反映在所有添加的位置上。为了避免这种问题，你应该在循环内部创建新的字典对象。

if __name__ == '__main__':  
  
    dicList = [{"a": 1, "b": 2, "c": 3}, {"a": 5, "b": 6, "c": 7}]  
    newList = []  
    for i in dicList:  
  
        for num in range(1, 3):  
            new_dict = {}  
            new_dict.update(i)  
            new_dict['d'] = 6+num  
            new_dict['e'] = 9+num  
            newList.append(new_dict)  
    print("新的数据")  
    for j in newList:  
        print(j)
