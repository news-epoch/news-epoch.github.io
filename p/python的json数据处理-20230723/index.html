title: Json的读取和写入
```Python
import json
# 读取json数据
json.load(数据)
# 往文件中写入json数据
json.dump(数据,文件对象,ensure_ascii=False)


```
- ensure_ascii=True：默认输出ASCLL码，如果把这个该成False,就可以输出中文。
- json.dumps() 是把python对象转换成json对象的一个过程，生成的是字符串。
- json.dump() 是把python对象转换成json对象生成一个fp的文件流，和文件相关。

离线保存

pickle模块 

	持续化模块：就是让数据持久化保存。
	pickle模块是Python专用的持久化模块，可以持久化包括自定义类在内的各种数据，比较适合Python本身复杂数据的存贮。
	但是持久化后的字串是不可认读的，并且只能用于Python环境，不能用作与其它语言进行数据交换。

作用：把 Python 对象直接保存到文件里，而不需要先把它们转化为字符串再保存，也不需要用底层的文件访问操作，直接把它们写入到一个二进制文件里。pickle 模块会创建一个 Python 语言专用的二进制格式，不需要使用者考虑任何文件细节，它会帮你完成读写对象操作。用pickle比你打开文件、转换数据格式并写入这样的操作要节省不少代码行。

背景 

	在爬虫开发过程中遇到这么一个问题，登录后获取到cookie，不想程序关闭后，cookie就消失了，需要重复使用，就需要将获取到的cookie原样离线保存到本地文件。

主要方法

在pickle中dumps()和loads()操作的是bytes类型，而在使用**dump()和lload()**读写文件时，要使用rb或wb模式，也就是只接收bytes类型的数据。

  1. pickle.dump(obj, file)：将Python数据转换并保存到pickle格式的文件内。
1
2

with open('data.pickle', 'wb') as f:
    pickle.dump(data, f)
  1. pickle.dumps(obj)：将Python数据转换为pickle格式的bytes字串。
1
2
3
4

import pickle
dic = {"k1":"v1","k2":123}
s = pickle.dumps(dic)
print(s)
  1. pickle.load(file)：从pickle格式的文件中读取数据并转换为Python的类型。
1
2

with open('data.pickle', 'rb') as f:
    data = pickle.load(f)
  1. pickle.loads(bytes_object)：将pickle格式的bytes字串转换为Python的类型。
1
2
3
4
5

import pickle
dic = {"k1":"v1","k2":123}
s = pickle.dumps(dic)
dic2 = pickle.loads(s)
print(dic2)

title: 离线读取
```python
# 示例一
cookies = pickle.load(open(filePath, "rb"))
for cookie in cookies:  
    if isinstance(cookie.get('expiry'), float):  
        cookie['expiry'] = int(cookie['expiry'])  
    self.chromebro.add_cookie(cookie)
# 示例二
filePath = os.path.join(self.BASE_DIR, 'data.pickle')  
data = []  
for i in pickle.load(open(filePath, "rb")):   # 读取可能会出现空字符，所以要进行判断
    if isinstance(i, dict):  
        data.append(i)
```
