->：lambda的一个标识
{}：代表了当前方法体，重写方法要做的内容
===============================
举例：
public interface Calc{
	public abstract int add(int a,int b);
}

public static void show(int a,int b,Calc c){
	int add = c.add(a,b);
	System.Out.print(add);
}
public static void main(String[] args){
	int a = 10;
	int b = 20;
	show(a,b,(c1,b1)->{return c1+b1;});
	show(a,b,(int a,int b)->{return a+b;});
	// 当方法体只有一句代码时，可以省略{}和return
	show(a,b,(c1,b1)->c1+b1);
}
