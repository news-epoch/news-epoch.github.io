        List<String> list = new ArrayList<>();  
        list.add("a");  
        list.add("b");  
        list.add("c");  
        list.add("d");  
        list.add("x");  
        System.out.println("-----------增强for循环-----------");  
        ArrayList<String> strings = new ArrayList<>();  
        list.forEach(s -> {  
            if ("x".equals(s)) {  
                strings.add(s);  
            }  
        });  
        list.removeAll(strings);  
        System.out.println("list = " + list);
    }
}

public class test01 {  
  
    @Test  
    public void test01(){  
        List<String> list = new ArrayList<>();  
        list.add("a");  
        list.add("b");  
        list.add("c");  
        list.add("d");  
        list.add("x");  
        System.out.println("-----------iterable-----------");  
		Iterator<String> iterator = list.iterator();
		while(iterator.hashNext()){
			if(iterator.next().equal("x")){
				iterator.remove();
			}
		}
		System.out.println("list = " + list);
    }
}

title: 通过迭代器进行循环抽取无需不唯一的集合
```Java
Connection c1 = new ArrayList();
Iterator iterator = c1.iterator();

while (iterator.hasNext()){
	Obejct ob = iterator.next();
	System.put.println(obj);
}
```
- 集合中自带iterator多态方法，该方法会创建一个迭代器对象，通过这个对象来实现迭代器功能。
	- hashNext() 判断游标/光标后面是否还存在数据
	- next() 拿到指定数据

5 集合工具类Collections

Java.util.Collections是一个集合工具类，用于操作LIst，Set，Map等集合。 Collections类提供了一系列的静态方法，可以实现对集合元素的排序，添加一些元素，随机排序，替换等操作。 并且Collections类是再集合的基础上进行修改，不会创建新的列表来进行接收。
